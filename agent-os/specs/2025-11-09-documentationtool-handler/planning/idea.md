# Raw Idea: DocumentationTool Handler

Create handlers for H5P.DocumentationTool content type (standalone and embedded). H5P.DocumentationTool is a content type that allows learners to create structured documentation with goal setting, planning, and reflection sections.

Following the Handler Development Guide patterns:
- Standalone-first architecture (DocumentationTool is runnable)
- Manual handler (DocumentationToolHandler) for YAML/JSON input
- AI handler (AIDocumentationToolHandler) for AI-generated content
- AIPromptBuilder integration with Universal AI Configuration
- Comprehensive validation and testing
